SafeReturn
==========

CMS developed for Fairfield PD and deployed in July 2013.  For background story, please see the press release:

http://fpdct.com/site/pressreleases/159-fairfield-police-launch-safe-return-network.html

Site was developed using C#, ASP.NET, Bootstrap 2, and an MSSQL database. Script to create the database has not been included, but may be added in the future.
